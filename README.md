### Built With

* Vue.js

### Prerequisites

* npm
```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo
```
git clone https://github.com/PeterGSWhite/typescholar.git
```
2. Install NPM packages
```
cd typescholar
npm install
```
3. Start the app
```
npm run dev
```
